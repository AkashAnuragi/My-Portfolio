# **App Name**: Visiofolio 3D

## Core Features:

- Animated Hero Section: Create a captivating hero section with a 3D animated background using CSS 3D transforms and animations. The intro text is presented with smooth entrance animations, drawing visitors into the portfolio immediately.
- Interactive Floating Objects: Implement interactive floating objects (geometric shapes or tech elements) within the 3D space that dynamically respond to mouse movement. This feature adds a layer of engagement and showcases technical skill.
- 3D Bio and Skills Showcase: Display a concise bio, highlighting key skills with interactive cards that subtly rotate or flip on hover. The personal photo will be presented with a soft 3D effect to enhance visual appeal.
- Dynamic Project Cards: Showcase projects using 3D-style cards that elegantly reveal project descriptions, technologies used, and demo links upon hover or click. Implement smooth transitions and ensure a responsive layout that adapts seamlessly to mobile, tablet, and desktop devices.
- Interactive Resume Timeline: Present education and work experience in an interactive timeline format that encourages user exploration. Offer a downloadable PDF button for the resume to facilitate easy access and sharing.
- Animated Contact Form: Design an animated contact form, enhanced with social media icons and engaging hover effects. The form should be intuitive and visually appealing, encouraging user interaction and feedback.
- Sticky Navigation with Smooth Scrolling: Implement a sticky top navigation bar that ensures easy access to all sections of the portfolio. Incorporate smooth scrolling functionality to create a seamless and fluid user experience when navigating between sections.

## Style Guidelines:

- Primary color: A vibrant, tech-inspired blue (#4285F4) to convey innovation and modernity.
- Background color: A desaturated light gray (#F5F5F5) to ensure readability and a clean aesthetic.
- Accent color: An energetic yellow (#FFCA28), analogous to the primary color, but different enough to make UI elements pop.
- Body and headline font: 'Inter', a sans-serif font known for its modern, machined look and excellent legibility, used both for headlines and body text.
- Use a set of minimalist, line-based icons to complement the modern design. These should be used for navigation, skills, and social media links.
- Employ a clean, grid-based layout to ensure responsiveness and consistency across all sections. Use spacing and padding to create a balanced and visually appealing structure.
- Incorporate subtle, fast-loading animations for transitions, hover effects, and scrolling to enhance user engagement without compromising performance.
